KERNEL PARA NOTEBOOK DE VIDEO (Text-to-Video e Image-to-Video)

1) Crear entorno virtual
python3 -m venv .venv-video
source .venv-video/bin/activate
python -m pip install --upgrade pip

2) Instalar librerias base
pip install diffusers transformers accelerate safetensors
pip install imageio imageio-ffmpeg matplotlib numpy pillow ipywidgets
pip install jupyter ipykernel

3) Registrar kernel en Jupyter
python -m ipykernel install --user --name video-diffusion --display-name "Python (video-diffusion)"

4) Verificacion rapida
python -c "import torch; print('torch', torch.__version__, 'cuda', torch.cuda.is_available())"
